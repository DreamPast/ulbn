#  ulbn: 高精度数字库

[English](./README.md) [简体中文](./README_zh_CN.md)

## 特色

- C89/C++98兼容（也有可选的C++20头文件）
- 只依赖于很少的硬件假设
- 规避了绝大部分的UB行为
- 严格的边界检查
- 可使用自定义的内存分配器
- 大部分函数进行了一定优化
- 无外置依赖

## 依赖

项目文件均无外部依赖

### ulbn.h和ulbn.c

最低C89/C++98，利用宏对部分代码进行优化建议

硬件假设：

- 要求`CHAR_BIT`是偶数

### ulbn.hpp

依赖C++20的以下特性：

- 约束与概念
- 三路比较
- \<bit\>头文件（假定机器为大端/小端）

### test

依赖以下对于现代平台的假设：

- 以补码形式存储有符号整数
- double的有效位数为52（IEEE754标准）
- char为8比特，且存在int64_t
